## 某些题目注释

### GCD

求最大公约数其实是一个二阶递推公式，其中：
$$
x_{n} = x_{n-2}\,\%\,x_{n-1}\quad(x_n \neq 0,n \geq 3)
$$


所以可以用数列的方法来写[1029-求最大公约数.cpp](exp5\1029-求最大公约数.cpp)
