# 一些坑，注意点

首先这个不同于数组（从 $0$ 开始计数），顺序表也好链表也好都是从 $1$ 开始计数的



## 链表

### 遍历

链表因为有头结点，所以在**遍历**的时候记得

```c++
link p = head->next;
```

### insert 和 del

然后无论是增加还是删除第 $i$ 个元素记得找第 $i-1$ 个元素进行操作。



这时候就要在while条件上面下功夫了，只需要根据

```c++
while(p != nullptr && count < i-1)
```

跟for是差不多

但是后面要判断一下

```c++
if(p == nullptr){
	return ERROR;
}
```

防止这个指标比原来的长度还要大，太离谱了

#### del

del肯定涉及了释放内存的操作

在c++中建议用这个

```c++
del node;
```

不要用

```c
free(node)
```
